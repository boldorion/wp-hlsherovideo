# Elementor HLS Video Plugin

> **Note:** This plugin is currently in beta and should be used cautiously on production sites.

The Elementor HLS Video Plugin is a WordPress plugin that enhances the functionality of Elementor by checking if a video on a page is an HLS video and loading the necessary JavaScript to play it correctly. This plugin aims to provide a seamless experience for users who want to incorporate HLS videos into their Elementor-powered websites.

## Features

1. **Video Detection:** The plugin checks whether an Elementor video is present on a page.
2. **HLS Video Support:** It loads the required JavaScript to determine if the video is in HLS format.
3. **Video Tag Update:** If the video is identified as an HLS video, the plugin updates the video tag to ensure proper playback of the video.

## Installation

To install the Elementor HLS Video Plugin, follow these steps:

1. Download the plugin ZIP file from the [GitHub repository](https://github.com/boldorion/wp-hlsherovideo).
2. Login to your WordPress admin panel.
3. Navigate to **Plugins** -> **Add New**.
4. Click on the **Upload Plugin** button.
5. Choose the downloaded ZIP file and click **Install Now**.
6. Once the installation is complete, click **Activate Plugin**.

## Usage

1. After activating the plugin, edit a page using Elementor.
2. Add a video widget to the page and configure the video source.
3. Save the changes and preview the page.
4. The plugin will automatically detect if the video is in HLS format and update the video tag accordingly.
5. If the video is HLS, it will play correctly on compatible browsers.
